# Modern Bank Application 🏦

A professional-grade banking management system that combines financial efficiency with Artificial Intelligence. This application provides users with a seamless interface to manage accounts, track transactions, and generate analytical financial reports.

## 🌟 Project Vision & AI Integration
This project was designed to go beyond traditional banking tools. By integrating **Artificial Intelligence algorithms**, the application can analyze transaction history, provide smart financial insights, and forecast future balance trends, helping users make better-informed financial decisions.

## 🚀 Key Features
- **Smart Dashboard:** Real-time account monitoring with an intuitive Dark Mode UI.
- **AI-Powered Analytics:** Automated data analysis to detect spending patterns and project future financial status.
- **Professional Reporting:** Generate high-quality, branded annual reports in PDF format.
- **Secure Data Handling:** Built-in persistence using SQLite for reliable account management.

## 🛠 Tech Stack & Libraries
This project leverages the power of Python's ecosystem to deliver a robust performance:

* **[CustomTkinter](https://github.com/TomSchimansky/CustomTkinter):** Used for creating a modern, high-DPI aware, and aesthetically pleasing User Interface.
* **[ReportLab](https://www.reportlab.com/):** The engine behind generating professional PDF financial reports.
* **[Matplotlib](https://matplotlib.org/):** Utilized for data visualization, allowing users to see their financial trends through interactive charts.
* **[SQLite3](https://docs.python.org/3/library/sqlite3.html):** A lightweight, file-based database for secure and efficient record-keeping.
